# Incursa.ProtocolLab.Model

`Incursa.ProtocolLab.Model` packages the shared ProtocolLab domain model
surface for downstream consumers.

## What it contains

- artifact layout and result-model types
- report serialization helpers
- shared model types used by adapters and conformance helpers

## What it is not

- the runner
- the CLI
- a hostable execution service

See the public repository docs for the broader ProtocolLab boundary:

- [ProtocolLab README](../../README.md)
