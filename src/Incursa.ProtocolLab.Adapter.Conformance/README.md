# Incursa.ProtocolLab.Adapter.Conformance

`Incursa.ProtocolLab.Adapter.Conformance` packages shared conformance helpers
for ProtocolLab adapter and test executor implementations.

## What it contains

- schema and contract validation helpers
- conformance utilities for adapter implementations
- conformance utilities for test executor implementations
- shared checks that keep public control-plane contracts honest

## What it is not

- the runner
- an adapter host
- a hosted validation service
