# Support

ProtocolLab is a public specification and contract repository. Support here is
limited to public contracts, schemas, fixtures, scenarios, suites, load
profiles, SpecTrace records, and documentation.

## Public Questions

Use GitHub issues for:

- public contract questions
- schema or fixture problems
- documentation corrections
- SpecTrace traceability questions
- scenario, suite, or load-profile proposals

Do not use public issues for private runner operations, hosted lab incidents,
private diagnostics, retained artifacts, credentials, or implementation-specific
debugging.

## Security Issues

Report suspected vulnerabilities privately through [SECURITY.md](SECURITY.md).

## Conduct Issues

Report conduct concerns through [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).

## Internal Or Implementation Support

Internal runners, hosted labs, package materialization, dashboards, provider
integrations, and publication pipelines are owned by implementation
repositories, not this public repository.
