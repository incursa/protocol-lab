# Incursa.ProtocolLab.Adapter.Contracts

`Incursa.ProtocolLab.Adapter.Contracts` packages the shared adapter
control-plane contracts used by ProtocolLab adapter implementations.

## What it contains

- adapter request and response contracts
- schema-facing types for adapter orchestration
- shared contract shapes used by conformance tooling

## What it is not

- an adapter host
- the runner
- a benchmark target

See the public repository docs for the broader ProtocolLab boundary:
