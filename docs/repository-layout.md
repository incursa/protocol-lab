# Repository Layout

The public ProtocolLab repository is organized around contracts and
declarative examples.

| Path | Purpose |
| --- | --- |
| `README.md` | Public entry point and repository scope summary |
| `LICENSE` | Apache-2.0 license |
| `SECURITY.md` | Security reporting policy |
| `CONTRIBUTING.md` | Contribution rules for the public contract repository |
| `CONTRIBUTOR-AGREEMENT.md` | Contributor agreement |
| `CHANGELOG.md` | Human-readable change notes |
| `.editorconfig` | Editor formatting hints for documentation and data files |
| `.gitignore` | Repository ignore rules |
| `.github/` | Issue and pull request templates only |
| `docs/` | Supporting documentation and boundary guidance |
| `docs/architecture/` | Public architecture notes for contract roles, scenarios, run plans, load, measurement, artifacts, and reports |
| `docs/audits/` | Repository audit records |
| `docs/lab/` | Package and run-plan contract documentation |
| `docs/protocol-lab/` | Product boundary, vision, roadmap, and readiness notes |
| `docs/reports/` | Public report, publication bundle, handoff, and safety documentation |
| `docs/scenarios/` | Scenario authoring and catalog documentation |
| `docs/migration/` | Boundary records for implementation assets moved out of the public repository |
| `specs/requirements/protocol-lab/` | SpecTrace specification artifacts |
| `specs/architecture/protocol-lab/` | SpecTrace architecture artifacts |
| `specs/work-items/protocol-lab/` | SpecTrace work-item artifacts |
| `specs/verification/protocol-lab/` | SpecTrace verification artifacts |
| `specs/traceability/` | Traceability guidance and convenience index |
| `schemas/` | JSON Schema and control-plane contract files |
| `fixtures/public-contracts/` | Declarative valid, invalid, and incompatible examples |
| `scenarios/` | Public scenario definitions |
| `suites/` | Public suite definitions |
| `load-profiles/` | Public load-profile definitions |

Implementation code, runnable automation, generated outputs, local validation
tools, hosted lab operations, and package publication workflows are out of
scope for this repository.
