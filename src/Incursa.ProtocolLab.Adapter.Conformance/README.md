# Incursa.ProtocolLab.Adapter.Conformance

`Incursa.ProtocolLab.Adapter.Conformance` packages shared adapter conformance
helpers for ProtocolLab adapter implementations.

## What it contains

- schema and contract validation helpers
- conformance utilities for adapter implementations
- shared checks that keep public adapter contracts honest

## What it is not

- the runner
- an adapter host
- a hosted validation service
